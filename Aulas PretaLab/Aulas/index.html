Bora?

O grande pai da web foi o físico Tim Berners Lee. Em 1989, Lee criou uma linguagem de marcação de hipertexto para construir páginas no ambiente virtual. Em 1990, ele criou com sucesso a primeira comunicação entre cliente HTTP e servidor usando essa linguagem. Em 1991, Tim criou a primeira versão da HTML, inicialmente projetada para conectar e compartilhar documentos entre instituições de pesquisa próximas.
